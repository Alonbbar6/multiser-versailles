# Multiser Versailles

Landing page for Multiser Versailles, Inc. — a Little Havana auto repair shop
(brakes, tune-ups, oil changes, general repairs).

Built with React + Vite + Tailwind CSS + DaisyUI + Framer Motion, following
the brand identity & design system (Ignition Red / Garage Teal / Deep Steel
palette, Inter + DM Serif Display typography).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via `.github/workflows/deploy.yml`.
